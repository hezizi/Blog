### 前言
#### 谈谈我为什么会写这篇Blog
>工作有一年多时间了，之前也偶尔写一写blog在CSDN中，然而并没有坚持，主要是执行力并不强，在sf和掘金中看文章时间久了，深深意识到写blog真的很重要，便于自己后续查看回顾，也可以帮助到更多人。所以在今年3月份（2018年）在CSDN中继续写blog（期间大约间隔有几个月没写），当时写的是`深入JavaScript之原型和原型链`，获得了一些赞，其实这对我还是有比较大的鼓舞的，到后续把blog地址迁移到github，，在今年4月份通过hexo搭建了个人博客，正式开始我的blog之旅。
